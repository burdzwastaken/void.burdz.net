## About

minimalist, monospace personal landing page with a simple blog system. the design focuses on simplicity, readability and a terminal-inspired aesthetic

## Features

- minimalist monospace design
- dark mode support
- simple blog system with Markdown support
- zero tracking or analytics
- terminal-inspired styling
- keyboard-friendly navigation

## Tech Stack

- HTML/CSS for structure and styling
- minimal yavascript for blog functionality
- markdown for writing blog posts that no one will ever read
- github pages for hosting
- github actions for deployment

## Local Development

```bash
make serve
```

## Creating Blog Posts

```bash
make new-post
```

add the new post to the `posts` array in script.js to make it visible (good reminder to myself)
